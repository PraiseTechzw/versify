# **App Name**: Versify

## Core Features:

- Image Capture & Upload: Allow users to upload images from their device or capture new ones using the camera, with auto image optimization and support for multiple formats and resolutions.
- AI-Powered Poem Generation: Generate unique poems based on image content, considering objects, colors, emotions, and offering multiple poetry styles and tone selection. The AI tool determines themes, emotions, and objects present in the photo.
- Poem Display & Editing: Display the generated poem alongside the image with options to edit, rewrite lines with AI suggestions, regenerate variations, and add an AI-generated title.
- Save, Organize & Share: Allow users to save poems and images in a personal library, create collections, export in various formats, and share to social media.
- Creative Controls: Provide creative controls like poem length slider, literal vs symbolic interpretation toggle, narrative choice, and keyword emphasis.
- Personalization: Offer user profiles with saved preferences, favorite styles, author signature, multi-language support, and optional daily inspiration notifications.
- AI Insights: Provide 'Behind the Poem' mode showing detected emotions and visual elements that inspired specific lines, along with a mood visualization meter.
- Flutter Integration: Develop cross-platform mobile application using Flutter framework.

## Style Guidelines:

- Primary Color: Deep Forest Green #228B22 — calm, grounded, contemplative
- Background Color: Soft Cream #F5F5DC — warm, open, and inviting
- Accent Color: Olive Green #6B8E23 — highlights, buttons, focus states
- Headline Font: Alegreya - Elegant serif for titles, poem headers, and emphasis
- Body Font: PT Sans - Clean, warm sans-serif optimized for long reading sessions
- Nature-inspired minimal icons (leaves, sun, moon, water, wind)
- Subtle fade-in and slide animations: When poems are generated, On scroll, On section transitions
